Sobre

Aplicação TotalityCRM 1.0.0

Aplicação desenvolvida para gerenciar clientes, produtos e cursos. O sistema permite o cadastro, edição e exclusão de clientes, produtos e cursos. O sistema foi desenvolvido com Python, Flask, SQLAlchemy, SQLite, HTML, CSS e Bootstrap.
